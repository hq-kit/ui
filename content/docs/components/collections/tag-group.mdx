### Links
Incorporate a hyperlink within the tag group using the `href` prop.
<Demo component='collections/tag-group/tag-group-href-demo' />


<Note intent="warning">
  For client-side routing, React Aria recommends using the <code className="font-semibold">onAction</code> event to handle navigation programmatically instead of passing an href prop.
  This is because some collection items cannot be rendered as native elements due to HTML specification limitations.
</Note>
