## API Reference

### FileTrigger

The `FileTrigger` component opens the system file picker from a styled trigger.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `variant` | `"default" \| "outline" \| "ghost" \| "destructive" \| "secondary" \| "link"`        | `"default"` |
| `size`    | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` |
| `isDisabled` | `boolean` | - |
| `isPending` | `boolean` | - |
