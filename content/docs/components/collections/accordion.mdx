## Examples

### Basic

A basic accordion that shows one item at a time. The first item is open by default.

<Demo component='collections/accordion/accordion-basic' />

### Multiple

Use `allowMultipleExpanded` to allow multiple items to be open at the same time.

<Demo component='collections/accordion/accordion-multiple' />

### Disabled

Use the `isDisabled` prop on `AccordionItem` to disable individual items.

<Demo component='collections/accordion/accordion-with-disabled' />

### Borders

Add `border` to the `Accordion` and `border-b last:border-b-0` to the `AccordionItem` to add borders to the items.

<Demo component='collections/accordion/accordion-with-borders' />
