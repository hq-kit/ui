## Respect Screen
By default, the menu will respect the screen size, when it lower than 768px, it will be displayed as a modal slide from the bottom. You can disable this behavior by setting `respectScreen` to `false`.
<Demo center component='collections/menu/menu-respect-screen-demo' />

## Sub Menu
<Demo center component='collections/menu/menu-sub-demo' />
<Note>
    When `respectScreen` is set to true (which is default), the menu renders in a drawer, but it only
    supports one submenu. If you try to open a second submenu, the modal will close, so use this
    feature with caution.
</Note>
