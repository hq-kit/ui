## Basic

A tag group is a focusable collection of labels, categories, keywords, filters, or other elements, equipped with keyboard navigation, selection, and removal functionalities.

<Demo center component='collections/tag-group/tag-group-demo' />

## Installation

<Install component='tag-group' />

## With Remove Button

Tags within the group can include a removal button.

<Demo center component='collections/tag-group/tag-group-with-remove-demo' />

## Variants

The default variant is `primary`. You can also use `secondary`, `success`, `warning`, and `danger`.

<Demo center component='collections/tag-group/tag-group-variant-demo' />

## Controlled

Manage the values of the tag group through the `value` prop.

<Demo center component='collections/tag-group/tag-group-controlled-demo' />

## Href

Incorporate a hyperlink within the tag group using the `href` prop.

<Demo center component='collections/tag-group/tag-group-href-demo' />

## Shape

The tag group can be shaped to be a square or a circle. Just like the badge component, see [badge](/docs/components/statuses/badge?#shape) documentation for more information.

<Demo center component='collections/tag-group/tag-group-shape-demo' />

## Disabled

Disable tags either by setting the `isDisabled` prop or using `disabledKeys` on the `TagGroup`.
