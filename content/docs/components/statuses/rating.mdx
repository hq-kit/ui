## Rating

Rating is basically a slider component that can be used to show user reviews and testimonials in the form of stars, reviews, likes, and more.

<Demo center component='statuses/rating-demo' />

## Installation

<Install component='rating' />

## Shapes

The shape of the rating can be set to `star`, `circle`, `square`, `diamond`, `triangle` or `heart`.

<Demo center component='statuses/rating-shapes-demo' />

## Specific Value

The max value of the rating can be set with the `maxValue` prop.

<Demo center component='statuses/rating-specific-value-demo' />

## Controlled

You can control the rating value with the `value` and `onChange` prop.
