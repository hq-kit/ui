## Sub Menu

Menu can be nested to display submenus.

<Demo center component='collections/menu/menu-sub-demo' />
<Note>
    When `respectScreen` is set to true (which is default), the menu renders in a modal, but it only
    supports one submenu. If you try to open a second submenu, the modal will close, so use this
    feature with caution.
</Note>

## Respect Screen

By default, the menu will respect the screen size, when it lower than 600px, it will be displayed as a modal slide from the bottom. You can disable this behavior by setting `respectScreen` to `false`.

<Demo center component='collections/menu/menu-respect-screen-demo' />

## Disabled

Disable specific menu items.

<Demo center component='collections/menu/menu-disabled-demo' />
Also, you can disable items directly in `Menu.Content` by using the `disabledKeys` prop.

```tsx
<Menu.Content disabledKeys={['gsu']} />
```
