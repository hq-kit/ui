## Installation
<Install component='toast' />

First, in your root app component or layout, import the Toast component.
```tsx
import { ToastProvider } from '@/components/ui'

const Layout = ({ children }: { children: React.ReactNode }) => {
    return (
        <>
            <ToastProvider />
            {children}
        <>
    )
}
```
